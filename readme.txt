enZombie Survival v1.0
https://community.7daystodie.com/topic/24594-enzombies-more-zombie-variations/

This add-on mod alters various game elements to tailor it to my personal playstyle - where it is more challenging and focused on survival. The enZombies mod is require to be installed for this survival mod to function.

Note: I also play with no traders on the map and instead install the enZombies Traders mod to spawn basic wandering traders.
